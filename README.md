# 🛡️ Port-Scanner-Shield (v3.3.3)

**Port-Scanner-Shield** is a professional-grade network reconnaissance tool developed to identify active service ports and assess the security posture of target systems. Built for **BSc in CSE** academic research and network auditing, this version features a modern web-driven dashboard for seamless interaction.

---

## 🚀 Key Features
- **Concurrent Scanning Engine**: Utilizes multi-threading to scan critical cybersecurity ports with extreme speed.
- **Dual-Mode Visuals**: 
  - **Hacker-style Black Mode** for low-light security operations.
  - **Professional Light-Gray Mode** for standard reporting and documentation.
- **Real-time Terminal Logs**: Integrated console view that streams live scan updates and discoveries.
- **Dynamic Port Mapping**: Scans industry-standard ports including HTTP, HTTPS, SSH, FTP, and Database services.
- **Smart UX**: Fully responsive design with **Enter Key** trigger support and instant status reporting.

## 🛠️ Tech Stack
- **Backend:** Python 3.x / Flask Framework
- **Network Module:** Socket Programming & Threading Architecture
- **Interface:** HTML5, CSS3 (Dynamic Theme Variables), JavaScript (Fetch API & DOM Manipulation)
