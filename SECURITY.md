# Security Policy | Port-Scanner-Shield

This security policy defines how vulnerability reports are handled for the **Port-Scanner-Shield (v3.3.3)**. 

## Supported Versions
| Version | Supported          |
| ------- | ------------------ |
| 3.3.3   | :white_check_mark: |
| < 3.0   | :x:                |

## Reporting a Vulnerability
If you discover a security flaw in this reconnaissance engine, please do not open a public issue. Instead, report it privately via my GitHub profile's contact methods. 

I aim to respond within 48 hours to any security-related inquiries to maintain the integrity of this academic portfolio project.
